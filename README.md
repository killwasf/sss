# WebSocket Server for MT5 Copy Trading

## Deploy to Render

1. Create account at render.com
2. Click "New +" -> "Web Service"
3. Connect repository or upload files
4. Configure:
   - Environment: Node
   - Build: npm install
   - Start: npm start
5. Deploy!

## Local Testing
```bash
npm install
npm start
```

Access: http://localhost:10000
